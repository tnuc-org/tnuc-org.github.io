Ventcheck done. Feature Requests?

Jun 30 2012

Well fuck me, looks like I did write some code after all. The new Ventcheck is pretty much done.
This is your once-in-a-lifetime chance for feature requests I might actually implement.
Anything? Email me.
(Don't worry, min view is still there, the screenshot is a bit older)

New features so far:
- A quick list to save servers
- Hovering over a channel highlights all users currently in that channel in the connections table
- Hovering over a user (connections table) highlights the channel he's in (see screenshot for those).
- Switching between min and normal view without having to reload the page



Since Ventrilo.com IP banned my server I rewrote everything in JavaScript and will release it as:
- A Google Chrome Extension
- A Firefox Plugin
- A Bookmarklet

The bookmarklet will probably work in Opera, MIGHT even work in IE9 (but not IE8 and below), but I won't bother testing in those. You shouldn't be using Internet Explorer anyway. Get a real browser and stop making the life of every web developer hell on earth.

PS:
I can't seem to find a public server that allows phantoms, so I can't realiably test those.
Got one that I could use for an hour (not logging in, just you setting up some)? Get in touch.

That fucking MSN Update.

Jun 22 2012

Last night Microsoft made the MSN/Windows Live 2011 update mandatory on Vista and Windows 7.
After updating and raging my face off I discovered that you won't be forced or even prompted to update on Windows XP, so all you have to do to get around updating is to start MSN in compatibility mode for Win XP SP3.

If you don't know how to do that, take a look at this quick guide.
