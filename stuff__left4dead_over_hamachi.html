Left4Dead / Left4Dead2 over Hamachi

back

Before scrolling down, it looks harder than it really is.
You can get through this entire wall of text within 2 minutes of explaining on vent instead.

I. Preparing L4D2


1. Install Left4Dead2.
3. Rightclick your L4D2 desktop shortcut and go to 'Properties'.
4. Add ' -console' at the end of the target path on the 'Link' tab. **
5. You might want to add ' -novid' to the path as well to suppress the intro video.
 (Example: "C:\Program Files\L4D2\l4d2.exe" -console)
6. Apply changes.

<**
If that's not possible because of a certain launcher, and the console won't open in the main menu:
-Start a single player game via the campaign, open the console ingame and type disconnect before hosting a game later.

If your (non-English) keyboard doesn't recognise the default (~) console hotkey:
-Open "Left 4 Dead 2\left4dead2\cfg\config.cfg" in a text editor that supports Linux linebreaks (NOT notepad).
-You'll notice it's alright when you see the single commands underneath each other, not on one line.
-Add the following line (feel free to replace "p" with any other unoccupied key/letter):
bind "p" toggleconsole
-Save and exit
Or download this config file instead (L4D2 default + console on [P])
**>


II. Configuring Hamachi:


1. Install Hamachi.
2. Launch it and follow the tutorial if you are not familiar with Hamachi.
3. Click the gear wheel icon in the lower right corner and go to 'Preferences'.
4. Go to the 'Window' tab and check 'Show advanced... peer menu item'.
5. Apply changes and close the Preferences window.
6. Connect to your L4D2 network.

Now for the annoying part:
Source based games require everybody in a LAN game to be in the same class C network. Means, if your Hamachi IP is 5.138.91.64 the IPs of everybody else have to start with 5.138.91. as well. The workaround is creating VPN Aliases, made-up IPs inside your own class C network telling the game everything is alright while they actually point at completely different IPs. Don't worry, you don't have to understand that, really.
Just take a look at these two screenshots:





Note:
Your Hamachi window will look different as I use custom labels.

My IP is shown in the main window: 5.138.91.64.
Kev's IP (Hamachi ID) is 5.141.237.38.
Now, to create a VPN alias you rightclick the user of your choice and go to 'Advanced'.
Leave his 'Peer VPN ADDRESS' as it is.
Set his 'Peer VPN ALIAS' to an IP of your choice using the first 3 parts of your own IP (here: 5.138.91.).
Use a random suffix between .1 and .254 (here: 44) EXCEPT FOR:
-Your own or
-One that is already used by somebody else

Additionally, set 'Local forwarding' and 'Remote forwarding' to 'enabled'.


Note:
If you are the hoster of a game, you have to do this for all players beforehand.
If you join somebody else's game, you only have to do this for the hoster of the game.
Ideally, have everbody do it for everybody in the network.

Now have a second look at the first screenshot:
Erik for example uses a random IP and the Alias 5.138.91.200 the ending of which is random but easy to remember as he is our default hoster. I suggest you use custom labels as well (rightclick > set label > custom) so you can look up others' aliases without having to go to the 'Advanced' user menu. But now back to hosting and actually playing the game.


III. Hosting a game

First-time setup:
1. Go to your control panel > 'Network Connections'.
2. Go to 'Advanced' > 'Advanced Settings'.
3. Select Hamachi and move it to the top of the list using the green arrows on the right.
4. Reboot your computer.





Hosting the game (finally):
1. Log off Steam, otherwise nobody will be able to connect without being on Steam also.
2. Start Left4Dead.
3. Type the following into the console.
(replace <<hamachi ip<< with your Hamachi IP (here: 5.138.91.64)
and <<map>> with the map name (eg. l4d_vs_hospital03_sewers or c3m4_plantation
note that l4d has different maps for coop (l4d_) and versus (l4d_vs_)) 

sv_lan 1
sv_allow_lobby_connect_only 0
ip <<hamachi ip>>
mp_gamemode coop|versus|survival           (leave out if coop)
                                           (disabled in l4d2 demo without SourceMod)
net_start
map <<map>>

4. Done, game should be loading by now.


IV. Joining a game

1. Start Left4Dead.
2. Type the following into the console.
(replace <<alias>> with the 'Peer VPN ALIAS' you set for the hoster (here: 5.138.91.200 if Erik hosts))

connect <<alias>>

3. Done, you should be connecting by now.
