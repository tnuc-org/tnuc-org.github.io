How to MC MSN / Windows Live Messenger

back

Many people have two separate MSN accounts, and open both WLM and Windows Messenger, the shitty one that comes with Windows XP, in order to be on both accounts at the same time. Windows Messenger doesn't support file transfer, games, cam, microphone though, so it gets fairly annoying at times. Luckily there's a workaround to this. By creating a value in the registry we can make WLM allow opening more than one instance and thereby be on several WLMs at the same time.

Start regedit (start menu > run > "regedit" OR [Win]+[R] > "regedit")
Navigate to the one of these paths:
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows Live\Messenger              (on 32bit Windows)
HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Microsoft\Windows Live\Messenger  (on 64bit Windows)


In the right half of regedit, rightclick the blank space, then select 'New' and 'DWORD' ('DWORD 32-bit' on x64 Systems).
Name the newly created DWORD 'Multiple Instances'
Rightclick it and select 'Modify'
Set the value to 1 (either hex or decimal, '1' is the same in both notations) and click 'Ok'

Done, you can now open as many instances of MSN/WLM as you want.
