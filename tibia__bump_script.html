BUMP

back

BUMP is a Tibia.com auto-poster.
Give it an account with a post char and it:
-posts a single reply (ie. to bump a single thread),
-spams replies every 32 seconds (ie. to be a massive dick),
-creates a new thread on all world boards or
-creates a new thread on all trade boards

I created it for personal use, so there's no debug output telling you what went wrong - if something went wrong.
Feel free to contact me for help though.

Requirements:
PHP5+ with cURL
Cronjob (optional)

Setup:
-Download and extract the rar
-Open it in a text editor (ie. Notepad)
-Enter your account and post data, uncomment the function call of choice (self-explanatory in source)
-Upload the .PHP file to your server
-Create and upload a text file named curltmpx.txt with CHMOD 666 to the same folder **
-Run the script manually or set up a cronjob

** for cURL cookies. Will be auto-created if PHP already has CHMOD 666 for the containing folder.

Options and Settings:
static $accname     = '1234567';     // Tibia Account Name, always needed.
static $password    = 'p@$$w0rD';    // Tibia Account Password, always needed.
static $charname    = 'My Char';     // Char Name, always needed unless you specify a $cid.
static $text        = "Hello World"; // Always needed, \n for linebreak (ie. "Te\nxt").
static $title       = '';            // Thread/Post subject, optional in bump/post mode.
static $threadid    = '1234567';     // The Thread ID. Only needed for bump/post mode.
static $cid         = '';            // The Character UID, leave blank if unknown (default).
                                     // Use this instead of a name if charname has ' or äöü.
                                     // Using cid will save one HTTP request per script call.
static $spam_rep    = 9;             // Posts per script call (spam mode)
