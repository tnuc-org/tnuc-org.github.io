This is my mapfiles.

If you don't know where to put them, go to:
Start > Run > type "%APPDATA%\Tibia"
Press [Enter] or Click OK and a folder will open.

Move the .rar File there and unrar it.
Make sure you select 'Extract here', NOT 'Extract to Automap'.
When prompted to overwrite, click 'Yes, all'.
